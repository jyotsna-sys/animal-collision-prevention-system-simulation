# Animal-Aware Collision Prevention System Simulation

A Python-based simulation developed using Pygame that demonstrates the software workflow of an animal-aware collision prevention system. The simulation visualizes vehicle movement, dynamic animal movement, distance monitoring, warning stages, speed adjustment, and audio-visual alerts in a real-time environment.

## Features

- Real-time vehicle movement simulation
- Dynamic animal movement
- Distance measurement between vehicle and animal
- Four-stage safety system:
  - Normal
  - Warning
  - Caution
  - Emergency
- Automatic speed reduction in caution mode
- Audio alerts for different safety stages
- Visual safety indicators and distance zones
- Emergency warning display
- Automatic simulation reset

## Technologies Used

- Python
- Pygame
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python animal_collision_sim.py
```
